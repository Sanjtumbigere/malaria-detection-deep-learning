# Malaria Detection Using Deep Learning

## Project Overview
This project focuses on detecting malaria-infected blood cells from microscopic images using deep learning techniques.

## Problem Statement
Manual detection of malaria is time-consuming and error-prone. This project automates the detection process using CNN-based models.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib

## Model Used
- Convolutional Neural Network (CNN)
- MobileNetV2

## Dataset
Microscopic blood smear images of infected and uninfected cells.
